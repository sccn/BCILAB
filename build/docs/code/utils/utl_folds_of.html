utl_folds_of

PURPOSE

Get the number of folds for a given cross-validation scheme.

SYNOPSIS

function result = utl_folds_of(scheme)

DESCRIPTION 

 Get the number of folds for a given cross-validation scheme.
 For 'loo', this is assumed to be ~10.
