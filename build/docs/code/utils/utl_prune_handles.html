utl_prune_handles

PURPOSE

Prune unreferenced workspace variables from anonymous functions in the given argument.

SYNOPSIS

function x = utl_prune_handles(x)

DESCRIPTION 

 Prune unreferenced workspace variables from anonymous functions in the given argument.

 In:

 Out:
